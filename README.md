# WordPress Backup Manager

A starter WordPress plugin that creates simple backups of WordPress files and provides a backup admin page.

## Features
- WordPress admin menu
- Backup folder creation
- Basic file backup logic
- Clean plugin structure

## Skills shown
WordPress plugin development, PHP file handling, admin pages, security checks.

## Installation
1. Upload the folder to `wp-content/plugins/`.
2. Activate **WordPress Backup Manager** in WordPress admin.
3. Go to **Tools > Backup Manager**.
